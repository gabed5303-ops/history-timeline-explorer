# History Timeline Explorer

This repository contains a simple web application built with Python and Flask that allows users to explore major events from the Cold War and modern history. Users can search for events by keyword, year, category, or key terms and view details about significant historical events.

## Features

- Search events by title, category, summary, or year.
- View events sorted chronologically.
- Responsive design using basic CSS.
- Easy to extend with additional events or categories.

## Getting Started

### Prerequisites

- Python 3.x installed on your machine.
- Flask (listed in `requirements.txt`).

### Installation

1. Clone this repository or download the source code.

   ```bash
   git clone https://github.com/gabed5303-ops/history-timeline-explorer.git
   cd history-timeline-explorer
   ```

2. Install the dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:

   ```bash
   python app.py
   ```

4. Open your web browser and navigate to `http://127.0.0.1:5000` to use the app.
